# DIMPLOTViewer
-----

Summary

-----
DIMPLOTViewer is a Python-based one-shot generator that converts DIMPLOT (LigPlot+) .drw files
into a self-contained interactive HTML viewer with embedded SVG graphics.
The tool is designed for post-export inspection and sharing of protein–protein interaction
diagrams generated by DIMPLOT, without requiring additional software or server-side components.

-----

Key features

-----
- Single-file HTML output (opens in any modern browser; no server required)
- Hybrid-resolution visualization
  - Residue-level interaction graph with residue-to-residue contact edges
  - Selective atom-level diagrams for residues involved in distance-labeled hydrogen bonds
- Semantic visualization
  - Residue coloring by physicochemical class
  - Hydrogen bonds rendered as dashed atom-to-atom lines with distance labels (Å)
- Interactive exploration
  - Drag residues to rearrange layout
  - Pan and zoom directly in the browser
  - Fit and Reset controls
- Chain separation
  - Chain A and Chain B separated by a boundary line
  - Chain-side lock (Split Y-lock) keeps Chain A above and Chain B below the boundary
- Local rotation
  - Alt + drag on focus residues rotates atom-level diagrams while preserving interaction geometry

-----

Requirements

-----
- Python 3.8 or newer
- No external Python dependencies (standard library only)
- Any modern web browser for viewing output (Chrome, Edge, Firefox, Safari)

-----

Installation

-----
1) Download the repository:

   git clone https://github.com/Yongjin0207/DIMPLOTViewer.git
   cd DIMPLOTViewer

-----

Quick start

-----
Convert a .drw file to an interactive HTML file:

   python3 DIMPLOTViewer.py INPUT.drw --out OUTPUT.html

Open the HTML file in a browser:

   Windows (PowerShell): start OUTPUT.html
   macOS: open OUTPUT.html
   Linux: xdg-open OUTPUT.html

-----

Usage

-----
Basic usage:

   python3 DIMPLOTViewer.py INPUT.drw --out OUTPUT.html

Common options:

--out OUTPUT.html
    Output HTML file name (default: DIMPLOTViewer.html)

--width N, --height N
    Output canvas size (default: 1600 x 900)

--margin N
    Margin in pixels around the layout (default: 80)

--atom_scale X
    Atom-level diagram scale factor (default: 3.0)

--split_extra X
    Extra chain separation in DRW units (default: 35.0)

--x_expand X, --y_expand X
    Global layout spreading around the centroid (default: 1.50, 1.00)

--relax_iters N
    Anti-overlap relaxation iterations (default: 320)

--relax_gap_px X
    Minimum extra gap between nodes in pixels (default: 14.0)

--relax_step X
    Relaxation step size (default: 0.38)

--relax_on_pointerup
    Run relaxation after pointer release (off by default)

--simple_r X, --focus_r X
    Node radii for simple residues and focus residues (default: 20.0, 44.0)

--boundary_extend_px X
    Extend the boundary line beyond the canvas width (default: 500.0)

--atom_label_mode element | atomname
    Atom label display mode (default: atomname)

--show_atom_labels_for_all
    Show labels for all atoms (off by default)

--chain_label_offset_px X
    Distance from boundary for Chain A and Chain B labels (default: 26.0)

-----

Example

-----
python3 DIMPLOTViewer.py example.drw --out example.html \
  --x_expand 1.6 --relax_iters 400 --relax_gap_px 16 --focus_r 50

-----

Interaction guide (in the browser)

-----
- Drag residues to move nodes
- Mouse wheel to zoom
- Shift + drag (or middle mouse drag on background) to pan
- Use Fit and Reset buttons to re-center the layout
- Alt + drag on focus residues to rotate atom-level diagrams

-----

Input format

-----
- Input must be a DIMPLOT .drw file
- Residue blocks and interaction records are parsed directly
- Contacts are de-duplicated at the residue-pair level for readability

-----

Output

-----
- A single self-contained HTML file with embedded SVG, CSS, and JavaScript
- Output can be shared and opened independently without additional software
